# 🎙️ 주소 AI 도슨트 (Address AI Docent)

우리 주변 길 이름에 숨겨진 이야기를 흥미롭게 풀어주는 다국어 AI 도슨트 애플리케이션입니다.

## 1. 프로젝트 개요
이 애플리케이션은 우리나라 도로명주소 뒤에 숨겨진 공식 유래 데이터를 바탕으로, Google Gemini AI가 다정하고 생동감 넘치는 '도슨트(해설사)' 역할로 대본을 생성하고, Edge TTS를 이용해 사용자에게 음성(다국어 지원)으로 이야기를 들려주는 웹 서비스입니다.

## 2. 주요 파일 설명

* **`app.py`**: 서비스의 메인 실행 파일입니다. 화면 UI 구성을 포함해 AI 텍스트 생성, 음성 합성 등 모든 핵심 로직이 들어 있습니다.
* **`road_names.json`**: 전국 시군구의 주요 도로명과 그 유래(부여사유)가 담긴 데이터 파일입니다. `app.py`에서 이 데이터를 읽어와 사용자 검색 기능에 활용합니다.

## 3. 핵심 기능 및 동작 원리

### ① 데이터 캐싱 및 결과 저장 (DB 로딩)
```python
@st.cache_data
def load_data():
```
* **동작 원리**: Streamlit의 `@st.cache_data` 데코레이터를 사용하여 `road_names.json` 파일을 최초 한 번만 디스크에서 읽어 메모리에 저장(캐싱)합니다.
* **DB 캐싱 (SQLite)**: 동일한 도로명과 언어로 도슨트 해설을 다시 요청할 경우, 매번 API와 TTS를 호출하는 대신 SQLite(`docent_cache.db`)에 저장된 스크립트와 로컬 오디오 파일을 즉시 반환하도록 최적화되어 있습니다. (성능 증대 및 API 비용 절감)

### ② AI 도슨트 스토리 생성 (`generate_docent_story`)
```python
def generate_docent_story(city, road, reason, api_key, target_lang="한국어"):
```
* **동작 원리**: Google의 `genai` SDK를 사용합니다. 사용자에게 입력받은 도로의 `공식 유래(reason)`를 기반으로, 방대한 역사적 배경지식을 결합하여 30초 내외의 친근한 대본을 작성하도록 체계적인 프롬프트 엔지니어링이 적용되어 있습니다. 
* **특징**: 단순 사실 나열이 아닌 '다정한 이야기꾼' 어조를 강제하며, 사용자가 선택한 언어(한국어, 영어, 중국어, 일본어)에 맞게 대본을 번역·생성합니다.

### ③ UI/UX 디자인 및 검색 기능
* **인터랙티브 웹 구현**: CSS (`st.markdown`) 디자인을 추가해 버튼 애니메이션 효과(`hover`)와 시각적으로 편안한 식물(Green) 계열 색상 테마를 구현했습니다.
* **기획 시리즈 (도슨트 큐레이션)**: 단순히 무작위의 길을 추천하는 것이 아니라, "🌟 도슨트 추천 길 이야기", "🌸 봄에 어울리는 길 이야기", "🏢 가장 흔한 길 이름 TOP 10" 등 10개 단위의 **테마(시리즈) 기능**을 사이드바에 구현했습니다. 이모지와 함께 구성된 시리즈를 클릭하면 즉시 화면에 추천 카드가 렌더링됩니다.
* **🗺️ 위치 시각화 (Google Maps iframe)**: 해당 도로의 위치를 화면 중간에 커다란 대형 지도(구글 맵)로 즉각 연동시켜 표시합니다.

### ④ 내 도슨트 도감 (게이미피케이션)
* **히스토리 기록**: DB(SQLite)를 활용해, 사용자가 AI로부터 해설을 넘겨받은 길들은 사이드바의 "📚 내 도슨트 도감" 버튼 목록으로 영구 저장됩니다. 
* **즉시 재생 (No API call)**: 도감의 길 버튼을 누르면 DB 캐시를 타게 되어 재생성 없이 바로 해설 오디오와 스토리를 불러오는 놀라운 응답 속도를 보여줍니다.

### ⑤ 다국어 TTS 음성 합성 및 파일 관리 (`generate_speech`)
```python
async def generate_speech(text, city, road, lang="한국어"):
```
* **동작 원리**: 마이크로소프트의 Edge TTS를 이용해 다국어 고품질 Neural 음성을 지원합니다.
* **에셋 관리**: 생성된 음성은 무지성으로 저장되는 것이 아니라, `mp3` 전용 폴더 내에 예쁜 이름(예: `구례군_산수유꽃길로_Korean.mp3`) 형식으로 저장되어 관리(다운로드 등)하기에 용이합니다.

## 4. 주요 기술 스택

* **Python 3.x**
* **프론트엔드/웹프레임워크**: Streamlit (`streamlit`)
* **LLM (텍스트 생성)**: Google Gemini API (`google-genai`)
* **TTS (음성 합성)**: Edge TTS (`edge-tts`)
* **비동기 처리**: `asyncio`

## 5. 실행 방법

1. 필수 라이브러리 설치
```bash
pip install streamlit google-genai edge-tts
```

2. (선택) 환경 변수에 Gemini API 키 등록 (코드에서 자동 로드됨)
```bash
# Windows
set GEMINI_API_KEY="본인의_API_키"
```
*(환경 변수가 등록되지 않은 경우 앱 좌측 ⚙️ 설정 사이드바에서 수동 입력 가능합니다.)*

3. 앱 실행
```bash
streamlit run app.py
```
